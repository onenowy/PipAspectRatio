# PipAspectRatio

Maintains aspect ratio when resizing Picture-in-Picture windows for KDE Plasma and KWin.

The "Keep above others" option is enabled by default for PiP windows.

## Installation

1. Clone this repository and Copy the folder to `~/.local/share/kwin/scripts/`
2. Open **System Settings** > **Window Management** > **KWin Scripts**
3. Enable **PIP Aspect Ratio** from the list

## Uninstallation

1. Disable the script via **System Settings** > **Window Management** > **KWin Scripts**
2. Remove the script folder:
   ```bash
   rm -rf ~/.local/share/kwin/scripts/PipAspectRatio
   ```
